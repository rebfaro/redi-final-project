# English Word Dictionary
This project is a web application that allows you to search for specific information about a English Word, such definitions, synonyms, antonyms, and more, using the WordsAPI.

## Features
• Word Search: Search any word to retrive data.
• Option Selection: Choose the type of information you want to get about the word (definitions, synonyms, etc.).
• Result Display: Shows the result in a simple list.

## Technologies Used
• HTML: Page structure.
• CSS: Page styling.
• JavaScript: Core logic, DOM Manipulation, and API calls.
• WordsAPPI: Thirs-party service for obtaining word data.

## How to Use
1. Dependencies: This code requires an HTML file with elements having the following IDs:
   • searchForm (the main form)
   • options (select dropdown)
   • display-word (to show the searched word)
   • myList (the unordered list for results)
   • display-results (the container for results)
2. API Key: The code uses a specif x-rapid-key. Replace it with your own API key from RapiAPI for WordsAPI.
3. Run: ipen the HTML file in your web browser
